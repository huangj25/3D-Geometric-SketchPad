# 3D Geometric SketchPad

一个基于SFML+OpenGL的3D几何画板工具，支持交互式3D物体查看。

- ## 操作说明
  
  | 操作  | 功能  |
  | --- | --- |
  | 鼠标左键拖拽 | 旋转物体 |
  | 鼠标右键拖拽 | 平移相机 |
  | 鼠标滚轮 | 缩放视野 |
  | 空格键 | 重置相机 |
  | ESC键 | 退出程序 |
  
  ## 技术栈
  
- SFML (窗口管理、事件处理)
- OpenGL + GLU (3D渲染)
- C++
  
  ## 编译运行
  
  确保已安装SFML和OpenGL开发库，编译后直接运行即可。编译器版本：MinGW GCC 13.2.0 64-bit
  
  ## 依赖
  
- SFML 2.6.2
- OpenGL
- GLU
